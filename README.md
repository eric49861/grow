## 介绍

使用非常牛逼的文档框架`docsify` 搭建个人网站，只需要写markdown就能搭建网站，再也不用在前端布局上花费太多的时间了。

## 该项目的目标

用于记录各种学习过程和成长历程，希望能够坚持写下去！！！

## 项目目录即相关文件介绍

- docs 顶级目录，项目的主要存放位置
  - algorithm 存放算法文档的目录
  - data-structure 存放数据结构的文档目录
  - .nojekyll 使用docsify创建项目时自动生成的
  - _coverpage.md 网站封面
  - _navbar.md 网站导航栏的定义
  - index.html 网站的入口文件
  - README.md 该网站的默认显示文档页面
- README.md 就是这个介绍文档
